Stylish

Submission for the Future Business Leaders of America Mobile Application Development Competition. The app was built using Meteor on Ubuntu Linux with Microsoft's Visual Studio Code IDE and Google's Android Studio build tools.

How to install for development

cd into the directory where this project is located and run the meteor install DEPENDENCY where DEPENDENCY are the following:

Dependencies

  • mdg:camera
  • twbs:bootstrap
  • accounts-ui
  • accounts-password
  • iron:router

Then run meteor remove autopublish as publishing is done manually. This will create an environment exclusively for Web development. To develop for mobile you must either be on Linux or Mac OSX. Meteor has tutorials for setting up an environment for Android development on Linux and Mac.
